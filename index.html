Analista de datos, aficionado a PowerBI, Base de datos y la Nube

Conoce a Luis Quispe

Soy un profesional altamente capacitado en el manejo de Microsoft Excel y PowerBI, con experiencia en la conexión y extracción de datos de SQL Server, SharePoint y PGSQL. Además, poseo habilidades en SQL (Server, Oracle y PG), así como en el uso de Google Cloud Platform (BigQuery, Google Cloud Storage). Mi dominio de PYTHON incluye bibliotecas como Pandas, Numpy y MatPlotLib, y también tengo experiencia en la creación de macros para realizar tareas como correos masivos, automatización y formularios. Con estas habilidades, puedo proporcionar soluciones efectivas y contribuir al éxito de cualquier proyecto o empresa en el análisis y gestión de datos.
